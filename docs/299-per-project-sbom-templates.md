# 299 — Per-Project SBOM Templates: Software Bill of Materials for agent supply-chain audit

**Anchors.** [270-agent-supply-chain-security](270-agent-supply-chain-security.md), [255-agntcy-oasf-acp-deep-dive](255-agntcy-oasf-acp-deep-dive.md), [273-agent-security-synthesis-2026](273-agent-security-synthesis-2026.md), [272-agent-compliance-and-audit](272-agent-compliance-and-audit.md). SLSA — https://slsa.dev/.

**One-line definition.** A **per-project SBOM (Software Bill of Materials) template** for the in-tree projects — extending classical software SBOM (CycloneDX / SPDX) with the **five agent-specific artifact classes** from [270](270-agent-supply-chain-security.md) (base model, MCP servers, marketplace skills, persistent memory partitions, A2A peers) — providing a YAML-shape template each project drops into `<project>/sbom.yaml`, auto-maintained by `harness_core/security/supply_chain/sbom.py`, consumed by the nightly revocation routine ([252](252-routines-pattern-for-self-hosted-agents.md)) for CVE / retraction matching, and serving as the technical-documentation evidence for EU AI Act Art. 11 + SOC 2 supply-chain controls.

## Auto-generation via `harness_core`

```python
# harness_core/security/supply_chain/sbom.py (skeleton)
class SBOMGenerator:
    def __init__(self, project_id: str, project_root: Path): ...

    def generate(self) -> dict[str, Any]:
        return {
            "sbom_version": "1.0",
            "generated_at": now_iso(),
            "project": self.project_id,
            "software_dependencies": self._scan_pip_npm_etc(),
            "base_models": self._scan_model_configs(),
            "mcp_servers": self._scan_mcp_configs(),
            "skills": self._scan_skill_manifests(),
            "memory_partitions": self._scan_memory_configs(),
            "a2a_peers": self._scan_a2a_clients(),
            "trust_summary": self._compute_trust_summary(),
        }
```

## Revocation routine integration

```python
# Daily revocation routine per docs/270
@routine(schedule="@daily", trigger=TriggerKind.CRON)
async def check_supply_chain_advisories(routine, trigger_kind, payload):
    sbom = load_sbom("/path/to/project/sbom.yaml")
    advisories = await fetch_advisory_feeds(["nvd", "smithery", "agntcy", "anthropic", ...])
    affected = match_advisories_to_sbom(advisories, sbom)
    for item in affected:
        await alert_oncall(item)
        if item.severity >= "CRITICAL":
            await auto_rollback(item)
```
